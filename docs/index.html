koa-angular-boostrap-sample

This is a sample project template using:

  • AngularJS

  • angular-ui-bootstrap

  • Koa@2.0

includes

  • gulp tasks

    • lint - javascript linter eslint module
    • test - nodejs testing mocha
    • coveralls - code coverage reports istanbul + Coveralls
    • pack - front end build system webpack
    • run - development run app nodemon

  • Koa routing using koa-load-routes

run

$ npm install

# start development server,
# nodemon will watch changes in your backend files to restart the server
# webpack will watch changes in frontend files to re compile
$ gulp serve
